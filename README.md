# Netflix EDA Project

## Project Overview

This project performs Exploratory Data Analysis (EDA) on the Netflix Titles Dataset to uncover trends, patterns, and insights related to movies and TV shows available on Netflix.

The analysis focuses on understanding content distribution, popular genres, country-wise contributions, ratings, release trends, and content growth over time.

---

## Objectives

* Analyze Netflix content using exploratory data analysis techniques.
* Identify trends in movies and TV shows.
* Examine content distribution across countries.
* Analyze ratings and genre popularity.
* Discover patterns in Netflix content growth.
* Generate meaningful business insights through data visualization.

---

## Dataset Information

Dataset: Netflix Titles Dataset

Features include:

* Title
* Type (Movie / TV Show)
* Director
* Cast
* Country
* Date Added
* Release Year
* Rating
* Duration
* Genre

## Key Insights

* Movies make up the majority of Netflix content.
* Netflix content has grown significantly over recent years.
* Certain countries contribute a large share of available titles.
* Drama and International content are among the most popular genres.
* Content additions increased rapidly during Netflix's expansion phase.
